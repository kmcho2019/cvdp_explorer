# Documentation Index

## Fast Navigation

- New to the repo:
  - `SPECIFICATION_AND_DEVELOPMENT_PLAN.md`
  - `ARCHITECTURE.md`
- Working on preprocessing/data shape:
  - `DATA_PIPELINE.md`
- Working on UI/UX and rendering:

- Running checks or CI troubleshooting:

- `CVDP_BENCHMARK_INTERNALS.md`: detailed `cvdp_benchmark` submodule internals and execution pipeline.
